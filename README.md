# java-fullstack-engineering

> Production-minded Java full-stack engineering for Spring Boot, SQL, and admin-style systems.

A Codex skill for Java full-stack engineering work across Spring Boot backends, SQL layers, and admin-style frontend integrations.

## What This Skill Does

This skill helps turn Java full-stack tasks into clean engineering execution. It is designed for real project work such as bug fixing, API cleanup, SQL tuning, refactors, project hardening, and interview-ready improvement.

## Best For

- Spring Boot bug fixes
- MyBatis or JPA persistence issues
- MySQL query cleanup and tuning
- Vue or React admin integration issues
- API contract cleanup
- turning student or side projects into stronger engineering evidence

## Inputs

Typical inputs:
- repository or module context
- framework and stack details
- bug report, logs, or failing behavior
- schema or API notes
- project-hardening goal for interviews or portfolio use

## Outputs

Typical outputs:
- focused code or config changes
- behavior explanation
- engineering rationale
- verification notes
- residual risks or follow-up suggestions

## Non-goals

This skill is not aimed at:
- purely visual frontend polish
- marketing copy or portfolio storytelling as the main task
- AI workflow architecture
- broad product brainstorming without code-level grounding

## Example Prompts

- `Help me fix this Spring Boot login bug and verify the API response shape.`
- `Refactor this Java backend service without changing the API contract.`
- `Optimize this MySQL query used by the recruitment recommendation page.`
- `Make this project more interview-ready without overengineering it.`
